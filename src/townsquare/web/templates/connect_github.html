Connect GitHub

townsquare uses your own GitHub personal access token. It only sees what your GitHub account can see — nothing more, nothing less.

How to generate a PAT

  1. Open github.com/settings/tokens
  2. Click Generate new token → Fine-grained (recommended)
  3. Resource owner: pick your work org if you want org-private repo access
  4. Repository access: choose what townsquare may search
  5. Repository permissions: Contents = read, Issues = read, Pull requests = read, Metadata = read
  6. Generate, copy the token (starts with github_pat_…)

Stored encrypted at rest. Only townsquare's federation router decrypts it during a query, and only to call GitHub on your behalf.
